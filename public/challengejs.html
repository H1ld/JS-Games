Challenge-js

Description

L’objectif est de créer une page web interactive permettant aux utilisateurs de jouer à différents jeux sur navigateur. 4 jeux et un algorithme au minimum sont requis :

  1. Snake
  2. Morpion
  3. Calculateur de vitesse de frappe au clavier
  4. Visual Memory Test
  5. Tri a bulle visuel

Spécifications techniques

    Langagas: HTML / CSS / JS – Bibliothèques autorisées mais framework (React, Angular …) interdits

  • Snake 🐍

    Le serpent doit être contrôlable à l’aide des flèches du clavier, il doit pouvoir manger des fruits répartis aléatoirement qui le font grandir lorsqu’il passe dessus. La partie doit se terminer si le serpent rencontre un mur ou s’il se heurte à lui-même.

  • Morpion ⭕❌

    Proposer deux modes de jeu, un mode 1 contre 1 ou les joueurs joueront chacun leurs tours et un mode joueur contre ordinateur. Pour le mode joueur contre ordinateur, l’ordinateur joue des coups aléatoires.

  • Calculateur de vitesse de frappe au clavier 🏁🏃

    Proposer un texte au joueur. Le joueur doit pouvoir voir son avancée sur le texte avec par exemple un curseur qui indique l’endroit du texte ou il se trouve. A la fin des 60 secondes, donner une moyenne de mots par minute de l’utilisateur.

  • Visual Memory Test 🧠

    Afficher un tableau de carrés avec certains allumés et d’autres non. Au bout de quelques secondes. Éteindre tous les carrés. L’utilisateur devra cliquer sur les carrés qui étaient allumés. Au fur et à mesure que les niveaux augmentent, augmenter le nombre de carrés total et le nombre de carrés allumés.

  • Tri a bulle visuel 🫧

    Implémenter l’algorithme du tri a bulle avec des nombres générés aléatoirement de façon à pouvoir voir le déroulé des différentes étapes du tri à l’écran.
